Platform Notes

This page collects Android and iOS differences, especially the ones most likely to affect integration results.

Audio routing and session restrictions

setDefaultAudioRouteToSpeakerphone and setAudioSessionOperationRestriction do not behave identically across platforms. Check platform behavior first when routing or session issues appear.

Channel option differences

publishThirdCameraTrack and publishFourthCameraTrack are Android/C++-side fields. iOS 4.5.3 does not expose equivalent fields directly. sourceType also needs explicit explanation in the iOS preview path.

Effect and parameter contracts

playEffect.gain currently follows an integer bridge contract. If effect playback looks wrong, verify the value before digging deeper.
