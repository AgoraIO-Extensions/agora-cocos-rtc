API Reference

This page collects the current public exports, client methods, and key events for exact lookup.

Top-level exports

createAgoraRtcClient

Create the main RTC client instance for almost every integration flow.

createAgoraRtcClient(options?: AgoraRtcClientOptions): AgoraRtcClient
Parameters
  • options.bridgeRuntime: optional custom bridge runtime.
  • options.timeoutMs: optional native request timeout in milliseconds. Default is 5000.
  • options.transport: optional explicit bridge transport.
Returns
  • Returns an AgoraRtcClient instance for later RTC calls.

AgoraRtcClient

on

Subscribe to SDK events.

on(eventName: K, listener: (payload: AgoraEventMap[K]) => void): () => void
Parameters
  • eventName: Event name from AgoraEventMap.
  • listener: Callback invoked when the event fires.
Returns
  • Returns a function that removes this listener.

off

Remove an SDK event listener.

off(eventName: K, listener: (payload: AgoraEventMap[K]) => void): void
Parameters
  • eventName: Event name from AgoraEventMap.
  • listener: Callback invoked when the event fires.
Returns
  • Removes the listener immediately.

Stats callback after leaving the channel.

on('leaveChannel', (payload: AgoraEventMap['leaveChannel']) => void)
Listener signature
  • Register this event through the client listener API.
Payload fields
  • duration: number — session duration in seconds.
  • txBytes?: number — optional total transmitted bytes.
  • rxBytes?: number — optional total received bytes.
  • txKBitRate?: number — optional total transmit bitrate in Kbps.
  • rxKBitRate?: number — optional total receive bitrate in Kbps.
  • txAudioBytes?: number — optional transmitted audio bytes.
  • rxAudioBytes?: number — optional received audio bytes.
  • txVideoBytes?: number — optional transmitted video bytes.
  • rxVideoBytes?: number — optional received video bytes.
  • txAudioKBitRate?: number — optional transmit audio bitrate in Kbps.
  • rxAudioKBitRate?: number — optional receive audio bitrate in Kbps.
  • txVideoKBitRate?: number — optional transmit video bitrate in Kbps.
  • rxVideoKBitRate?: number — optional receive video bitrate in Kbps.
  • lastmileDelay?: number — optional last-mile network delay.
  • cpuTotalUsage?: number — optional total CPU usage ratio.
  • gatewayRtt?: number — optional gateway round-trip time.
  • cpuAppUsage?: number — optional app CPU usage ratio.
  • users?: number — optional user count in the stats sample.
  • connectTimeMs?: number — optional connection establishment time in milliseconds.
  • txPacketLossRate?: number — optional transmit packet loss rate.
  • rxPacketLossRate?: number — optional receive packet loss rate.
  • memoryAppUsageRatio?: number — optional app memory usage ratio.
  • memoryTotalUsageRatio?: number — optional total memory usage ratio.
  • memoryAppUsageInKbytes?: number — optional app memory usage in KB.
