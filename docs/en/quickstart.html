Quickstart

The goal is the shortest path to a working video call: prepare the example, initialize the engine, bind a local view, join a channel, and confirm success through the key callbacks.

1. Prepare the example project

./scripts/prepare-example.sh

Then open example/basic-call in Cocos Creator 3.8.8.

2. Initialize, enable video, and join the channel

Here is the shortest path that actually renders a frame. Order matters: call setRenderBackend(...) before initialize(...).

import { createAgoraRtcClient } from '../../extensions/agora-rtc/js/agora.ts';

// These values come from your own backend; placeholders shown here.
const appId = 'YOUR_APP_ID';
const token = 'YOUR_TOKEN';
const channelId = 'test-channel';
const uid = 1001;

const client = createAgoraRtcClient();

// The render backend must be chosen before init.
await client.setRenderBackend('engine-texture');
await client.initialize(appId);

// Enable video and bind the local frame to a render region.
await client.enableVideo(true);
await client.setupLocalVideoView({ x: 0, y: 0, width: 320, height: 180 });

await client.joinChannel(token, channelId, uid);
createAgoraRtcClient setRenderBackend initialize enableVideo setupLocalVideoView joinChannel

If you prefer to browse by capability first, jump to Core APIs.

3. Verify success signals

  • The joinChannelSuccess callback fires, confirming the local user has joined the channel
  • With engine-texture rendering and a bound local view, you receive the localVideoTextureReady callback
  • The example's status panel keeps updating RTC events and call statistics

If texture views or mirror behavior are unclear, continue with Rendering.
