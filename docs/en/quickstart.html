Quickstart

The goal is to prepare the example, initialize the engine, join a channel, start preview, and verify the key callbacks in the shortest path.

1. Prepare the example project

./scripts/prepare-example.sh

Then open example/basic-call in Cocos Creator 3.8.8.

2. Initialize and join the channel

import { createAgoraRtcClient } from '../../extensions/agora-rtc/js/agora.ts';

const client = createAgoraRtcClient();
await client.initialize(appId);
await client.joinChannel(token, channelId, uid);

If you want the grouped capability view first, jump to Core APIs.

3. Verify success signals

  • joinChannelSuccess has fired
  • If you bind texture views, you can observe localVideoTextureReady
  • The example status area keeps updating RTC events and summaries

If texture views or mirror behavior are confusing, continue with Rendering.
