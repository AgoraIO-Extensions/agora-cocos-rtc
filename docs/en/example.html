Example

example/basic-call is not a glossy visual demo. It is a QA console used to validate integration flow, observe events, and probe public APIs quickly.

QA console role

The action panels and status output are organized around RtcSessionService so UI actions map back to public APIs and observable runtime state.

Grouped actions

  • Full Demo: runs the grouped capability probes in sequence
  • Channel: channel and role related probes
  • Mixing: audio mixing probes
  • Effect: audio effect probes
  • Diag: logging and diagnostics probes
