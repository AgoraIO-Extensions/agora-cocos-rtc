Build Agora RTC features in Cocos Creator

This doc set is for developers who are new to both Agora RTC and this Cocos integration. Start with the shortest successful path, then move into grouped capabilities and platform constraints.

What this repo ships

  • sdk/agora-rtc: the Cocos extension package, JavaScript wrapper, and native bridge templates
  • example/basic-call: a QA-oriented example project for integration checks and troubleshooting
  • Android and iOS 4.5.3 native bridge coverage plus the engine-texture render backend (video frames uploaded as Cocos engine textures)

What you need first

  • Cocos Creator 3.8.8
  • Node.js 18+
  • Android Studio / Android SDK / JDK 17
  • Xcode 15+ and CocoaPods
  • A valid Agora App ID

Recommended reading path

  1. Start with Quickstart to initialize, enable video, and join
  2. Read Core APIs to see capabilities grouped by integration task
  3. Use Rendering to choose a render backend and handle video frames
  4. Check Platform Notes for Android / iOS behavior differences
  5. Jump to API Reference for exact methods and callbacks
