Platform Notes

这一页集中解释 Android 和 iOS 的行为差异，但关键限制仍然要在相关 API 附近重复出现。

音频路由与会话限制

setDefaultAudioRouteToSpeakerphone 和 setAudioSessionOperationRestriction 在不同平台的行为不完全一致，文档需要明确说清楚，不要让开发者自己猜。

频道选项差异

publishThirdCameraTrack 和 publishFourthCameraTrack 是 Android/C++ 侧字段，iOS 4.5.3 并不对等暴露。sourceType 也需要在 iOS 预览路径里单独解释。

音效与参数约束

playEffect.gain 目前走整数桥接约定；这类契约必须写在平台说明和具体 API 两边。
