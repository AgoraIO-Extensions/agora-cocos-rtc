Example

example/basic-call 更像一个 QA 控制台，用来验证接入链路、观察事件和快速试 API。

QA 控制台定位

动作面板和状态面板围绕 RtcSessionService 组织，方便把 UI 操作映射回公开 API。

分组动作

  • Full Demo：依次跑完能力探测
  • Channel：频道和角色相关演示
  • Mixing：音频混音相关演示
  • Effect：音效相关演示
  • Diag：日志和诊断相关演示
