API Reference

这一页是精确查阅入口。条目必须和当前公开导出保持一致，并给出稳定锚点。

Top-level exports

createAgoraRtcClient

创建 RTC 客户端实例。

createAgoraEngineTextureViewManager

创建 engine-texture 视图管理器。

createAgoraEngineTextureViewController

创建 engine-texture 视图控制器。

getAgoraEngineTextureBridge

读取当前运行时的引擎纹理桥。

AgoraRtcClient

on

订阅事件。

off

取消订阅事件。

initialize

初始化 RTC 引擎。

joinChannel

使用数字 UID 加入频道。

joinChannelWithUserAccount

使用字符串账号加入频道。

getUserInfoByUserAccount

查询字符串账号对应的用户信息。

setRenderBackend

设置渲染后端。

setVideoEncoderConfiguration

设置编码参数。

setupLocalVideoView

建立本地视频视图。

setupRemoteVideoView

建立远端视频视图。

startPreview

开始本地预览。

stopPreview

停止本地预览。

switchCamera

切换摄像头。

startAudioMixing

开始音频混音。

playEffect

播放音效。

setParameters

设置扩展参数。

getEngineTexture

读取缓存的引擎纹理对象。

isEngineTextureReady

判断纹理槽是否已经 ready。

destroy

销毁客户端并清理事件与待处理请求。

Priority events

joinChannelSuccess

加入频道成功。

leaveChannel

离开频道后的统计回调。

rtcStats

RTC 统计信息。

localVideoTextureReady

本地纹理已准备就绪。

remoteVideoTextureReady

远端纹理已准备就绪。

renderBackendState

渲染后端状态变化。

error

统一错误出口。
