API Reference

这一页是精确查阅入口。条目必须和当前公开导出保持一致，并给出稳定锚点。

Top-level exports

createAgoraRtcClient

创建 RTC 客户端实例，是大多数接入流程的入口。

createAgoraRtcClient(options?: AgoraRtcClientOptions): AgoraRtcClient
参数说明
  • options.bridgeRuntime：可选，自定义桥运行时。
  • options.timeoutMs：可选，原生请求超时时间，默认 5000ms。
  • options.transport：可选，直接传入桥传输对象。
返回值
  • 返回一个 AgoraRtcClient 实例，后续通过它调用 RTC API。

createAgoraEngineTextureViewManager

创建 engine-texture 视图管理器，用于处理纹理槽和显示节点绑定。

createAgoraEngineTextureViewManager(client: AgoraRtcClient, bridgeRuntime?: CocosBridgeRuntime): Promise<AgoraEngineTextureViewManager>
参数说明
  • client：必填，已经创建好的 RTC 客户端。
  • bridgeRuntime：可选，自定义运行时桥上下文。
返回值
  • 返回 Promise，resolve 为 AgoraEngineTextureViewManager。

createAgoraEngineTextureViewController

创建 engine-texture 视图控制器，适合封装本地/远端视图注册逻辑。

createAgoraEngineTextureViewController(client: AgoraRtcClient): AgoraEngineTextureViewController
参数说明
  • client：必填，RTC 客户端实例。
返回值
  • 返回 AgoraEngineTextureViewController 实例。

getAgoraEngineTextureBridge

读取当前运行时中可用的引擎纹理桥，适合排查纹理槽状态。

getAgoraEngineTextureBridge(runtime?: CocosBridgeRuntime): CocosEngineTextureBridge | null
参数说明
  • runtime：可选，显式传入运行时对象。
返回值
  • 返回桥对象；若当前环境没有桥，则返回 null。

Priority events

joinChannelSuccess

加入频道成功后触发。

on('joinChannelSuccess', (payload: AgoraEventMap['joinChannelSuccess']) => void)
监听签名
  • 通过客户端事件监听接口订阅这个事件。
payload 字段
  • channelId：string，关联的频道名。
  • uid：number，数字用户 UID。
  • elapsed：number，相关状态切换的耗时。

leaveChannel

离开频道后的统计回调。

on('leaveChannel', (payload: AgoraEventMap['leaveChannel']) => void)
监听签名
  • 通过客户端事件监听接口订阅这个事件。
payload 字段
  • duration：number，字段含义取决于原生回调约定。
  • txBytes?：number，字段含义取决于原生回调约定。
  • rxBytes?：number，字段含义取决于原生回调约定。
  • txKBitRate?：number，字段含义取决于原生回调约定。
  • rxKBitRate?：number，字段含义取决于原生回调约定。
  • txAudioBytes?：number，字段含义取决于原生回调约定。
  • rxAudioBytes?：number，字段含义取决于原生回调约定。
  • txVideoBytes?：number，字段含义取决于原生回调约定。
  • rxVideoBytes?：number，字段含义取决于原生回调约定。
  • txAudioKBitRate?：number，字段含义取决于原生回调约定。
  • rxAudioKBitRate?：number，字段含义取决于原生回调约定。
  • txVideoKBitRate?：number，字段含义取决于原生回调约定。
  • rxVideoKBitRate?：number，字段含义取决于原生回调约定。
  • lastmileDelay?：number，字段含义取决于原生回调约定。
