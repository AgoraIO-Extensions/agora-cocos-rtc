先跑通最短链路

快速开始

目标是用最短路径跑通一条完整链路：准备示例、初始化引擎、加入频道、开启本地预览，并通过关键回调确认接入成功。

1. 准备示例工程

./scripts/prepare-example.sh

然后用 Cocos Creator 3.8.8 打开 example/basic-call。

2. 初始化引擎、开启视频并加入频道

下面是一条能出画面的最短链路。注意顺序：setRenderBackend(...) 必须在 initialize(...) 之前调用。

import { createAgoraRtcClient } from '../../extensions/agora-rtc/js/agora.ts';

// 这些值由你的业务服务端下发，这里仅作占位。
const appId = 'YOUR_APP_ID';
const token = 'YOUR_TOKEN';
const channelId = 'test-channel';
const uid = 1001;

const client = createAgoraRtcClient();

// 渲染后端必须在初始化前确定。
await client.setRenderBackend('engine-texture');
await client.initialize(appId);

// 开启视频并把本地画面绑定到一块渲染区域。
await client.enableVideo(true);
await client.setupLocalVideoView({ x: 0, y: 0, width: 320, height: 180 });

await client.joinChannel(token, channelId, uid);
createAgoraRtcClient setRenderBackend initialize enableVideo setupLocalVideoView joinChannel

只做语音通话时，可以省略 enableVideo 和 setupLocalVideoView。想先按能力分组看接口，可以跳到 Core APIs。

3. 确认接入成功

  • 收到 joinChannelSuccess 回调，说明本端已成功加入频道
  • 如果使用 engine-texture 渲染并绑定了本地视图，会收到 localVideoTextureReady 回调
  • 示例的状态面板会持续刷新 RTC 事件和通话统计信息

如果对纹理绑定或镜像行为还有疑问，继续阅读 Rendering。
