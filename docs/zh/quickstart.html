Quickstart

目标是在最短路径内完成准备示例、初始化引擎、加入频道、开始预览，并验证关键回调。

1. 准备示例工程

./scripts/prepare-example.sh

然后用 Cocos Creator 3.8.8 打开 example/basic-call。

2. 初始化并加入频道

import { createAgoraRtcClient } from '../../extensions/agora-rtc/js/agora.ts';

const client = createAgoraRtcClient();
await client.initialize(appId);
await client.joinChannel(token, channelId, uid);
createAgoraRtcClient initialize joinChannel

想先按能力分组看接口，可以跳到 Core APIs。

3. 验证成功信号

  • joinChannelSuccess 已触发
  • 如果绑定纹理视图，观察到 localVideoTextureReady
  • 示例状态区持续刷新 RTC 事件和统计信息

如果你不确定纹理视图或镜像行为，继续读 Rendering。
