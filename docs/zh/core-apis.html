核心 API

这一页不按源码文件拆分 API，而是按你在接入时实际要完成的事情来分组，方便你快速定位该调用哪个接口。

引擎与会话

创建客户端、初始化引擎、销毁释放资源，是每条接入链路的起点和终点。

createAgoraRtcClient initialize destroy

频道与身份

设置频道场景、加入和离开频道。数字 UID 与字符串用户账号两套身份体系任选其一。

setChannelProfile joinChannel joinChannelWithUserAccount getUserInfoByUserAccount

音频

本地采集与远端播放控制、麦克风开关、音量提示等基础音频能力。混音与音效见下方独立分组。

enableLocalAudio muteLocalAudioStream enableAudioVolumeIndication

视频与视图

设置视频编码属性，并把本地与远端画面绑定到 Cocos 渲染区域或纹理槽位。

setVideoEncoderConfiguration setupLocalVideoView setupRemoteVideoView

渲染与视觉控制

选择渲染后端、调整编码与镜像表现。入会前先用 setRenderBackend 确定渲染路径。

setRenderBackend setVideoEncoderConfiguration

混音、音效与诊断

背景音乐混音、短音效播放，以及日志与诊断相关接口。

startAudioMixing playEffect setLogFile

完整接口清单请查阅 API 参考。
