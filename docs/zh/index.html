在 Cocos Creator 中接入 Agora RTC

这套文档面向第一次接触 Agora RTC 和这个 Cocos 插件的开发者。先跑通最短链路，再按能力和平台差异继续深入。

仓库提供的内容

  • sdk/agora-rtc：交付给业务项目的 Cocos 扩展、JS API 包装和原生桥模板
  • example/basic-call：用于验证接入和排查问题的 QA 控制台示例
  • Android 和 iOS 4.5.3 桥接能力，以及 engine-texture 渲染路径

你需要准备什么

  • Cocos Creator 3.8.8
  • Node.js 18+
  • Android Studio / Android SDK / JDK 17
  • Xcode 15+ 和 CocoaPods
  • 有效的 Agora App ID

推荐阅读路径

  1. 先读 Quickstart 跑通初始化、入会和预览
  2. 再读 Core APIs 理解能力分组
  3. 遇到纹理和视图问题时读 Rendering
  4. 查精确方法时跳到 API Reference
